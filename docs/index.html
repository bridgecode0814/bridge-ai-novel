GitHub 시나리오 관리

작품과 에피소드를 GitHub 저장소에 저장하고 불러옵니다. 브랜치별로 버전을 관리할 수 있어요.

⚠️ 로컬 파일(file://)로 열려 있어요
브라우저 보안 정책으로 GitHub API가 차단됩니다. 아래 방법 중 하나로 실행하세요:
python -m http.server 8080
# 터미널에서 실행 후 → http://localhost:8080
또는 VS Code Live Server 확장 → 파일 우클릭 → Open with Live Server
💡 GitHub Personal Access Token이 필요합니다.
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
→ repo 권한 체크 후 생성하세요.
